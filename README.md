#Tech Stack
• Laravel 10 Breeze
• Tailwindcss

## Run Locally

Clone the project

```bash
  git clone https://github.com/Oecophyllaa/rentdrive.git
```
Install dependencies

```bash
php artisan breeze:install
php artisan migrate
```

Configure your database inside the .env file

```bash
  DB_DATABASE=laravel10-breeze
  DB_USERNAME=root
  DB_PASSWORD=secret
```

Start the server

```bash
  php artisan serve
```
